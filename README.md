# Expense App - Smart Expense Tracker

Expense App is a fast, resume-friendly expense tracker built with vanilla HTML, CSS, and JavaScript. It runs fully in the browser, saves data with `localStorage`, and includes realistic finance workflows instead of being a tiny CRUD demo.

## Features

- Add income and expense transactions with category, account, date, and notes.
- View balance, monthly income, monthly spending, and savings rate.
- Filter and search transactions by text, category, and type.
- Track category budgets with visual progress and warning states.
- Review recurring bills and savings goal progress.
- Analyze spending with a responsive canvas chart and insight cards.
- Import and export JSON data for backup or project demonstration.
- Responsive design for desktop, tablet, and mobile screens.

## Why this works well on a resume

- Shows practical product thinking: budgets, recurring expenses, goal tracking, analytics, and backup flows.
- Demonstrates DOM manipulation, client-side state management, forms, validation, storage, filtering, and canvas rendering.
- Uses no heavy framework, so the code is easy to explain as a beginner.
- Runs instantly by opening `index.html`, which makes demos simple.

## How to run

Open `index.html` in a browser.

## How to publish on GitHub Pages

1. Create a new GitHub repository named `expense-app`.
2. Upload `index.html`, `styles.css`, `app.js`, `README.md`, and the `assets` folder.
3. Go to repository `Settings`.
4. Open `Pages`.
5. Under `Build and deployment`, choose `Deploy from a branch`.
6. Select the `main` branch and `/root` folder, then save.
7. GitHub will give you a public website link.
